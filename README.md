# R_Miniproject_KoreaMeteorologicalAnalysis

주제 : 국내 기온, 강수량, 폭염을 이용한 데이터 분석 미니프로젝트

#6 조 프로젝트 기획서

  프로젝트명(가칭) : 국내 기온,강수량 분석

  프로젝트 개요

   - 2010 ~ 2020년도 기상청 데이터 기반의 국내 기온,강수량 비교
   - 2019 ~ 2021년도 기상청 데이터 기반의 지역별 폭염 빈도 분석

   
  프로젝트 주요 특징

    - 연도별 기온, 강수량 비교
    - 월별 기온, 강수량 비교
    - 지역별 기온, 강수량 비교
    - 계절별 기온, 강수량 비교 
    - 지역별 폭염 빈도수에 따른 텍스트 마이닝
